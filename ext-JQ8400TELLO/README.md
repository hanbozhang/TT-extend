## Summary
TT语音扩展模块使用JQ8400作为播放芯片，使用自带的一线串口发送命令，也可以使用DF或网上的JQ8400模块进行学习和调试，一线串口定义到TTESP32中的IO26，播放状态检测BUSY定义到IO13上。
